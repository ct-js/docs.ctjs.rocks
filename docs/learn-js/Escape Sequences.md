# Escape Sequences

```
*Code 	Result 	Description
*\' 	' 	Single quote
*\" 	" 	Double quote
*\\ 	\ 	Backslash

*the following may or may not work (not used in html)

*\b 	Backspace
*\f 	Form Feed
*\n 	New Line
*\r 	Carriage Return
*\t 	Horizontal Tabulator
*\v 	Vertical Tabulator
```
